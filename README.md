
# StockFlow - Sistema de Inventario

StockFlow es una aplicación web sencilla para la gestión de inventario
Permite registrar múltiples productos, especificar sus características y visualizar el inventario de forma organizada.

## Características

- Registro dinámico de productos
- Selección de unidades (tallas, líquidos, peso)
- Validación de datos
- Navegación con tecla Enter
- Interfaz moderna y responsive
- Visualización final del inventario

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Inter)

## Funcionamiento

1. El usuario ingresa la cantidad de productos a registrar.
2. Se genera automáticamente un formulario dinámico.
3. Se capturan los datos de cada producto.
4. Se validan los campos.
5. Se muestra el inventario final en una tabla.

## PROYECTO: STOCKFLOW

Descripción:
StockFlow es una página web que permite registrar productos en un inventario de manera dinámica
El usuario puede ingresar varios productos, elegir su tipo de medida y cantidad.

Funcionamiento:
Primero, el usuario ingresa el número de productos que desea registrar. Después
el sistema genera automáticamente los campos necesarios para capturar la información.

Cada producto incluye:
- Nombre
- Tipo de medida (talla, litros o peso)
- Cantidad

El sistema valida que todos los campos estén completos antes de guardar.

Al finalizar, se muestra una tabla con todos los productos registrados.

## Tecnologías usadas:
- HTML para la estructura
- CSS para el diseño
- JavaScript para la lógica

## Objetivo:
Facilitar el registro de inventario de forma rápida y sencilla.

## Conclusión:
El sistema cumple con su función básica y puede mejorarse agregando almacenamiento de datos y más funcionalidades.
